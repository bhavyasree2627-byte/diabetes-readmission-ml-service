# Assumptions

1. Readmission within 30 days is treated as the positive class and all other outcomes are treated as negative.

2. The selected numerical features provide a reasonable baseline representation of patient encounter characteristics.

3. The dataset is assumed to be de-identified and suitable for educational and research purposes.

4. Local CPU execution is sufficient for both model training and inference.

5. Random Forest provides an acceptable balance between performance and simplicity for this assignment.

6. The prediction service is intended for demonstration purposes and not for clinical decision-making.

7. Input requests to the API are assumed to follow the documented schema.
